# Showcase-Portfolio
This is a website that displays video made by yours truly. The purpose of this website is to show the world a collection of edits I made. 

##Background
I had been making edots for about a year at that point, while also doing school and I wanted to make things. So I decided why not combine the two things I enjoy doing and put it into one banner 'Creating'. At first it was really hard getting into it, problems like hosting and such were very relevant and discouraging but I have yet to figure them out. As of now I'll continue to create, while I figure it out.

I have a lot of ideas about this website, as for now I'll be contetnt with the way it functions.
